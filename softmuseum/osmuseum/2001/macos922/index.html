Mac OS 9.2 is a version of Mac OS. Released in June 18 2001, it increased the system requirements to G3 processor. It also improved performance and classic environment support. Mac OS 9.2.1 (August 21, 2001) and 9.2.2 (December 5 2001) are free updates for Mac OS 9.2. Both updates contains bug fixes.
