macOS

macOS (formerly OS X, Mac OS X, Mac OS and Macintosh System Software) is a graphical operating system developed by Apple designed for use with its Macintosh computers. First released in 1984 as "Macintosh System Software", and rebranded as "Mac OS" in 1996. In 2001, Mac OS X was released which brought an entirely new architecture based on NeXTSTEP.
